â–¶ï¸ Methods to Run

1. Windows Batch Script

run.bat

Automatically installs dependencies and starts server

2. PowerShell

.\run.ps1

Requires execution policy: Set-ExecutionPolicy Bypass

3. npm Commands

npm install          # Install dependencies
npm start           # Start with auto-open
npm run server      # Start server only
npm run dev         # Development mode with caching disabled

4. Python

python -m http.server 8000

Built-in Python web server

5. Direct Access

Open HTML files directly in browser (some features may not work)

ğŸ¨ Customization

Change Port Number

Edit package.json:

{
  "scripts": {
    "start": "http-server -p 3000 -o",
    "server": "http-server -p 3000"
  }
}

Customize Colors

Edit CSS variables in any HTML file:

:root {
  --primary: #165DFF;
  --primary-hover: #0E4BD9;
  --foreground: #080C1A;
  --secondary: #6A7686;
  --success: #30B22D;
  --error: #ED6B60;
}

Add New Font

Import in HTML head:

<link href="https://fonts.googleapis.com/css2?family=Your+Font&display=swap" rel="stylesheet">
